## Descrição

Sistema de monitoramento de satélite agrícola utilizando Python.

O sistema simula dados de telemetria de um satélite responsável pelo monitoramento de lavouras e gera alertas automáticos para auxiliar operadores do centro de controle.

## Tecnologias

* Python
* VS Code

## Funcionalidades

* Simulação de telemetria
* Monitoramento de NDVI
* Monitoramento de temperatura
* Controle de armazenamento
* Verificação de estabilidade orbital
* Geração automática de alertas

## Impacto na Terra

O AgroSat auxilia no monitoramento agrícola, permitindo acompanhar a saúde das plantações, identificar problemas em lavouras e melhorar a tomada de decisão no setor agrícola.

## Como executar

python main.py
